VIBE Standards Council / VSC-0001

VSC-0001 (Draft): Core VIBE Data Contract

1. Scope

This draft standard defines baseline structure for exchanging VIBE-related observations among participating tools and institutions.

2. Required Fields

  • vibrations: 0.0 to 1.0
  • intimacy: 0.0 to 1.0
  • bureaucracy: 0.0 to 1.0
  • entropy: 0.0 to 1.0

3. Compliance Note

Conforming systems must document measurement conditions and preserve source context when publishing composite scores.
